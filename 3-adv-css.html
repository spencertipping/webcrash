More selectors

  • h1 + h2: select h2's that come after h1's
  • p > *: all direct children of paragraphs
  • p[title]: all paragraphs whose "title" attributes are set
  • a[target="_blank"]: links opening in other windows
  • a[href^="ftp://"]: FTP links

Pseudo-classes

  • Represents a piece of an element, or a state
  • a:hover -- a link when you're hovering over it
  • *:focus -- anything focused
  • p:first-child -- a paragraph that is the first child of its parent
  • p:last-child -- a paragraph that is the last child of its parent

More Pseudo-classes

  • *:nth-child(5) -- the fifth element of every container
  • *:nth-child(2n) -- even elements
  • *:nth-child(2n+1) -- odd elements
  • *:nth-child(even) -- the right way to do even elements
  • *:nth-child(odd) -- ditto for odd elements

Generating stuff

  • :before, :after -- imaginary elements before and after things
  • content: 'string' -- a rule to put HTML into imaginary elements

Parts

  • p::first-line -- the first line of the paragraph
  • span::first-letter -- the first letter in the span
  • An article going over all this stuff

Extra goodies

  • opacity: x -- for x between 0 and 1, sets the opacity of an element
  • border-radius: x -- uses rounded corners
  • css3.info, describing many more cool things about CSS3
