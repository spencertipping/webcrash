What does jQuery do?

  • Grab a bunch of elements
  • Do something to each of them
  • For example, $('div').append('foo') appends 'foo' to every div
  • Try it out

What's going on?

  • jQuery gives you a function called $
  • $('string') matches elements selected by the string
  • $('string').method() does the same thing to all matched elements
  • (The $ function also has methods: $.get, $.post, $.makeArray, etc.)

Selectors

jQuery uses CSS-style selectors to grab elements.

Examples:
  • $('p'): select all paragraphs
  • $('p, div'): select all paragraphs and divs
  • $('#foo'): select the element with ID "foo"
  • Full reference

Common methods

All methods return the object they are called on.
  • addClass/removeClass: adds/removes a CSS class from matched elements
  • attr(name, value): sets an attribute on each matched element
  • append(stuff), prepend(stuff): appends or prepends some HTML to each matched element
  • hide(), show(): instantly makes elements invisible or visible

More common methods

  • css(property, value): sets a CSS property of all elements
  • click(callback): calls callback when any matched element is clicked
  • text(string): sets the text of each matched element to the string
  • Full reference

  • no page reloading

Then a client:

Starting with an empty jQuery app

Body elements

Application code

$('#update').click (function () {
  $.get('/date.php', function (date) {
    $('#time').text(date);
  });
});
