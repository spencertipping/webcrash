The body section

Common body elements

  • div: a box of stuff
  • span: a flowing wrapper of stuff
  • h1, h2, ..., h6: headers
  • a: a hyperlink wrapping stuff
  • img: an image

More body elements

  • ul: unordered list of items
  • ol: ordered list of items
  • li: a list-item wrapper of stuff
  • table: a table of tr nodes
  • tr: a table row of td or th nodes
  • th: a table header cell of stuff
  • td: a regular table cell of stuff

Form elements

  • input: a text box, password box, check box, radio button, etc.
  • select: a dropdown box of options
  • option: an option in a dropdown box
  • button: a clickable button
  • textarea: a large area of text

Tragic mistakes

  • object: a way to embed a plugin
  • embed: another way to embed a plugin
  • font: the reason we have CSS
  • iframe: a sub-page often used for advertising
  • form: AJAX that reloads the page

Common attributes

  • id: names an element uniquely
  • class: sets the CSS class of an element (more later)
  • style: sets the CSS style directly
  • href, src: specify a URL

Under the hood...

Everything is a node

<span id='myspan'>
  <!-- This is a comment -->
  Some text in the span
</span>
  1. The span containing everything else
  2. The comment
  3. The text

In HTML parlance...

  • The <span> node is the parent of the others (which are children)
  • The comment and text are siblings of each other
  • Children of children (and so on) are descendants
  • Parents of parents (and so on) are ancestors
  • Nodes that look like <word> are called elements
